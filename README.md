# CV — Kévin Dolié

CV web de Kévin Dolié — **Directeur Digital · Head of Digital · CTO** (Toulouse).

Page web autonome (HTML/CSS/JS, sans dépendance de build) au positionnement dirigeant :
direction d'équipes jusqu'à 150 personnes, pilotage de la production digitale en grand
groupe bancaire, co-fondation et direction d'une agence digitale (stratégie, P&L, croissance).

Design corporate exécutif — navy / or discret, typographie serif + sans-serif classique,
timeline de trajectoire comme fil conducteur.

## Contenu

- `index.html` — le CV en ligne (page web autonome, tout intégré).
- `assets/cv-print.html` — la source de la version imprimable une page.
- `assets/CV_Kevin_Dolie.pdf` — le CV téléchargeable (A4, une page), lié depuis le site.

## Consultation

Ouvrir `index.html` dans un navigateur, ou publier via GitHub Pages.

### Activer GitHub Pages

1. **Settings** → **Pages**
2. Source : branche `main`, dossier `/ (root)`
3. Le site sera disponible sur `https://kevindle1.github.io/Cv_kevin_dolie/`
   (URL vers laquelle pointent le lien canonique et le QR code du CV).
